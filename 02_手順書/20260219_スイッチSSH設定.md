# 手順
特権モード→設定モードへ
```bash
en
conf t
```


ポート①に 172.31.0.10/24
```bash
interface FastEthernet 0/3
 no switchport
 ip address 172.31.0.10 255.255.255.0
 no shutdown
 exit
```
ポート②に 172.31.1.10/24
```bash
interface FastEthernet 0/2
 no switchport
 ip address 172.31.1.10 255.255.255.0
 no shutdown
 exit
```

ルーティングを有効化（L3として動かす）
```bash
ip routing
```


SSHのための基本情報を入れる（ホスト名/ドメイン名）
```bash
hostname L3SW
ip domain-name lab.local
```


③ SSHログイン用ユーザを作る
```bash
username admin privilege 15 secret Password123
```


④ RSA鍵生成（SSHが有効になる設定）
```bash
crypto key generate rsa
```
聞かれたら：

```bash
How many bits? 2048
```


⑤ SSH v2 を指定
```bash
ip ssh version 2
```
確認：
```bash
show ip ssh
```


⑥ VTY(リモートログイン回線)をSSHのみにして、ローカルユーザで認証
```bash
line vty 0 4
login local
transport input ssh
exit
```


⑦ 設定保存
```bash
write memory
```


■接続テスト
```bash
ssh admin@172.31.0.10
ssh admin@10.0.0.10
```
