# スイッチ初期化手順
### **1. 特権モードへ**
```bash
enable
```
---
</br>


### **2. 設定削除**
```bash
write erase
```
確認が出たら：
```bash
Erasing the nvram filesystem will remove all configuration files! Continue? [confirm]
```
EnterでOK。

---
</br>


### **3. 再起動**
```bash
reload
```
聞かれたら：
```bash
Proceed with reload? [confirm]
```
Enter。
```bash
System configuration has been modified. Save? [yse/no]: no
```
※ もし保存しますか？と聞かれたら No

---
</br>


### **4. 再起動後**
```bash
Would you like to enter the initial configuration dialog?
```
→ `no`

でOK。

---
</br>


### **5. 初期化確認**
```bash
show running-config
```
ほぼ空になっていれば成功。
