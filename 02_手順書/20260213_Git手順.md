## 1. ローカルリポジトリ作成
まず、Gitに上げたいディレクトリへ移動します。
```bash
# cd /path/to/your/directory
cd ~/Documents/workspace
```
Git管理を開始します。
```bash
git init
```
▼ 実行結果
```bash
[kurosawamasaya@kurosawamasayanoMacBook-Pro workspace]$ git init
Initialized empty Git repository in /Users/kurosawamasaya/Documents/workspace/.git/
[kurosawamasaya@kurosawamasayanoMacBook-Pro workspace (main #)]$ 
```
▼ 確認
```bash
[kurosawamasaya@kurosawamasayanoMacBook-Pro workspace (main #)]$ ls -la
total 3
drwxr-xr-x@  8 kurosawamasaya  staff    256  2月 13 15:09 .
drwx------@ 14 kurosawamasaya  staff    448  2月 16 12:10 ..
drwxr-xr-x  12 kurosawamasaya  staff    384  2月 16 12:30 .git # ←が追加されること
[kurosawamasaya@kurosawamasayanoMacBook-Pro workspace (main #)]$ 
```
これで `.git` フォルダが作成され、ローカルリポジトリになります。

</br>

## 2. ローカルリポジトリに add / commit
**状態確認**
```bash
git status
```
**ステージング**</br>
すべて追加する場合：
```bash
git add .
```
個別に追加する場合：
```bash
git add ファイル名
```
**コミット**
```bash
git commit -m "初回コミット"
```

</br>

## 3. リモートリポジトリを作成（まだ無い場合）
GitHub で `新規リポジトリ作成` を行います。

1. GitHub にログイン
2. 「New repository」
3. Repository name を入力
4. README は作らない（重要）
5. Create repository

作成すると以下のURLが表示されます。
```bash
# https://github.com/ユーザー名/リポジトリ名.git
https://github.com/masayakurosawa-git/workspace.git
```

</br>

## 4. ローカルとリモートを紐付け(SSH接続)
ローカルリポジトリとリモートリポジトリを紐づけます。
```bash
# git remote add origin git@github.com:ユーザー名/リポジトリ名.git
git remote add origin git@github.com:masayakurosawa-git/workspace.git
```
確認：
```bash
git remote -v
```
▼ 実行結果
```bash
[kurosawamasaya@kurosawamasayanoMacBook-Pro workspace (main)]$ git remote -v
origin	https://github.com/masayakurosawa-git/workspace.git (fetch)
origin	https://github.com/masayakurosawa-git/workspace.git (push)
[kurosawamasaya@kurosawamasayanoMacBook-Pro workspace (main)]$
```

## 7. ブランチを作成する方法
**1. 現在のブランチ確認**
```bash
git branch
```
▼ `* main` のように `*` が付いているブランチが現在位置です。
```bash
* main
```
</br>

**2. 最新状態を取得（推奨）**
```bash
git fetch origin
git merge origin/main

# 以下のやり方でもOK
git pull origin main
```
</br>

**3. 作成＋同時に切替**
```bash
git checkout -b 新しいブランチ名
```
</br>

**4. ブランチ確認**
```bash
git branch
```
▼ 新しいブランチに `*` が付いていればOK。
```bash
main
* newbranch
```
</br>

**5. 作業 → add / commit**
```bash
git add .
git commit -m "作業内容"
```
</br>

**6. リモートへ反映（初回のみ）**
```bash
git push -u origin ブランチ名
```
2回目以降は
```bash
git push
```
だけでOK。
