### gitに手順書をアップロードする

## 1. ローカルリポジトリ作成
まず、Gitに上げたいディレクトリへ移動します。
```bash
cd /path/to/your/directory
```
Git管理を開始します。
```bash
git init
```
これで `.git` フォルダが作成され、ローカルリポジトリになります。

---
</br>
</br>

## 2. ローカルリポジトリに add / commit
**状態確認**
```bash
git status
```
**追加**

すべて追加する場合：
```bash
git add .
```
個別に追加する場合：
```bash
git add ファイル名
```
**コミット**
```bash
git commit -m "初回コミット"
```
---
</br>
</br>

## 3. リモートリポジトリを作成（まだ無い場合）
GitHub / GitLab / Bitbucket などで 新規リポジトリ作成 を行います。

例：GitHub の場合

1. GitHub にログイン
2. 「New repository」
3. Repository name を入力
4. README は作らない（重要）
5. Create repository

作成すると以下のURLが表示されます。
```bash
https://github.com/ユーザー名/リポジトリ名.git
```
---
</br>
</br>

## 4. ローカルとリモートを紐付け
```bash
git remote add origin https://github.com/ユーザー名/リポジトリ名.git
```
確認：
```bash
git remote -v
```
---
</br>
</br>

## 5. Push（初回）
```bash
git branch -M main
git push -u origin main
```
※ GitHub の初期ブランチが main のため合わせます。
