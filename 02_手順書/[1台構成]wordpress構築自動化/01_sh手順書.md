# 目的
### 同一EC2に以下をwordpressを構築し、動作するようにする。
</br>
</br>

# 構成
```bash
[ EC2① ] Web + PHP + WordPress 
```
</br>
</br>

# 手順
### 情報の定義
```bash
# 定義
EC2_IP="54.214.89.122"
KEY="$HOME/.ssh/CL_kurosawa_key.pem"
SH_FILE="$HOME/Downloads/lamp_wordpress_setup.sh"
```


### shファイル転送
```bash
# ファイル転送
scp -i "$KEY" "$SH_FILE" ec2-user@"$EC2_IP":/home/ec2-user/
```


### EC2に接続
```bash
# ssh接続
ssh -i "$KEY" ec2-user@"$EC2_IP"
```


### shファイルを実行する。
```bash
# 権限変更
chmod 755 lamp_wordpress_setup.sh

# シェル実行
./lamp_wordpress_setup.sh
```
