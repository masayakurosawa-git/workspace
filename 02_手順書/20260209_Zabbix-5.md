# 監視用のEC2に ZabbixAgent を構築する

## 1. stress のインストール
EC2接続
```bash
ssh -i ~/.ssh/CL_kurosawa_key.pem ec2-user@54.213.10.15
```


root権限に変更
```bash
sudo su -
```


Zabbix リポジトリの登録
```bash
dnf install -y stress
```
</br>
</br>


## 2. stress による負荷テスト
負荷スタート
```bash
stress -c 2
```


管理画面で確認
```bash
# http://<EC2 インスタンスの IP>/zabbix/
http://34.221.226.198/zabbix/
```
