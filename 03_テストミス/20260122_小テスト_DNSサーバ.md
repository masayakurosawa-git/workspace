# 間違えた問題
## 問題１
#### 自身でドメインのゾーン情報を管理・保持し、他のサーバに問い合わせることなく応答を返すことができるDNSサーバを何と呼びますか？

- a. レジストラ
- b. DNSキャッシュサーバ
- c. 権威DNSサーバ
- d. プライマリサーバ
</br>
### 正解
---------------------------
c. 権威DNSサーバ
</br>
</br>
### 解説
---------------------------
#### DNSキャッシュサーバ</br>
クライアントからのリクエストを受けて他のDNSサーバへ問い合わせ、結果をキャッシュする役割を持ちます。
</br>
</br>
#### 権威DNSサーバ</br>
IPアドレスとドメイン名の紐付けに関して、自分の管理する範囲内のIPアドレスとドメイン名の対応表を持っていて、問い合わせに対してはその対応表をもとに返事をするDNSサーバさんのこと

</br>
</br>
</br>


## 問題２
#### ドメイン宛のメールをどのサーバに配送するかを指定するためのDNSレコードは何ですか？

- a. MXレコード
- b. PTRレコード
- c. Aレコード
- d. SOAレコード
</br>
### 正解
---------------------------
a. MXレコード
</br>
</br>
### 解説
---------------------------
#### MXレコード</br>
権威DNSサーバさんがお仕事で使うネタ帳（ゾーンファイル）の中身
そのドメイン宛のメールの配送先メールサーバが書かれている行のこと
