# 間違えた問題
### `問題６`　Apacheで別のサイトへ画面遷移させるときに使う設定は何か
### -- 選択肢
a. systemctl redirect httpd </br>
b. 適切な回答はない </br>
c. apachectl configtest </br>
d. apachectl 301 </br>
e. DocumetRedirect </br>
f. apachectl redirect </br>


### -- 正解
b. 適切な回答はない
</br>

### -- 解説
#### Redirect ディレクティブ (mod_alias)
最も簡単で一般的な方法です。単一のページやディレクトリ単位でリダイレクトする場合に使用します。 
</br>

#### RewriteRule ディレクティブ (mod_rewrite)
より複雑な条件や、URLのパラメータを保持したままリダイレクトしたい場合に使用します。 
</br>

#### RedirectMatch ディレクティブ
正規表現を使って、より細かくリダイレクトルールを指定したい場合に使用します。
</br>

## 
### `問題８`　ApacheがTomcat と連携するためのプロトコルを答えよ。
### -- 選択肢
a. POP </br>
b. AJP </br>
c. SNMP </br>
d. RPC </br>
e. DNS </br>
f. JAVA </br>


### -- 正解
b. AJP
</br>

### -- 解説
#### AJP（Apache JServ Protocol）
`プロトコル名:`　AJP (Apache JServ Protocol)</br>
`通信モジュール:`　Apache側では主に mod_proxy_ajp モジュールが使用されます。</br>
`接続ポート:`　一般的にデフォルトで 8009 ポートが使用されます。</br>
`メリット:`　HTTPよりもバイナリ形式であるAJPの方が、フロントエンドWebサーバとバックエンドのアプリケーションサーバ間の通信において効率的（高速）であるとされています。</br>
