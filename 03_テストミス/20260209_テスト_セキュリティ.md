#### 感染すると攻撃者にコンピュータが乗っ取られ、知らず知らずのうちに別のコンピュータ/サーバへの攻撃などに使われるものは何と言いますか？
```
a. ウイルス
b. ボット
c. バックドア
d. トロイの木馬
```
```
正解: ボット
```
```
■ ボット（Bot）

意味：
感染したPCを遠隔操作できる「操り人形状態」のマルウェア。

ポイント：

ユーザーは気づかない

DDoS攻撃・迷惑メール送信などに使われる

多数集まると「ボットネット」になる
```

#### 正当なソフトウェアに見せかけて不正な動作を行うマルウェアは何ですか？
```
a. スパイウェア
b. トロイの木馬
c. アドウェア
d. ルートキット
```
```
正解: トロイの木馬
```
```
■ トロイの木馬

意味：
正規ソフトを装って侵入し、不正動作を行うマルウェア。

ポイント：

見た目は安全そう

インストールすると情報盗取やバックドア作成

「中に敵兵が隠れている」イメージ
```

#### 企業をターゲットにして、偽のビジネスメールを送り金銭を騙し取る詐欺は何ですか？
```
a. ビジネスメール詐欺（BEC）
b. スパムメール
c. フィッシング
d. マルウェア
```
```
正解: ビジネスメール詐欺（BEC）
```
```
■ ビジネスメール詐欺（BEC）

意味：
企業の担当者や上司になりすまし、送金させる詐欺。

ポイント：

「至急振込をお願いします」など緊急性を装う

実在の会社名・署名を使うため見抜きにくい

個人より企業が主な標的
```

#### 膨大なパスワード候補を機械的に試行し、正しいパスワードを見つけ出す攻撃手法は何ですか？
```
a. ブルートフォース攻撃
b. 辞書攻撃
c. スニッフィング
d. バッファオーバーフロー
```
```
正解: ブルートフォース攻撃
```
```
■ ブルートフォース攻撃

意味：
パスワードを総当たりで試す攻撃。

ポイント：

「1234」→「1235」→…と機械的に全部試す

桁数が短いほど突破されやすい

対策：長く複雑なパスワード＋アカウントロック
```
