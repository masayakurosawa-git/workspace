# NFSサーバの設定
### 共有ディレクトリの作成
```bash
mkdir /share
```

### exportsの設定
```bash
# ファイル編集
vi /etc/exports
-----------------
/share 172.31.0.0/16(rw,no_root_squash)
# {共有したいディレクトリ} {許可したいクライアント}({与えたい権限をカンマ区切りで})
# rw → 書き込み、読み込み権限
# no_root_squash → root権限を付与
-----------------

# 起動
systemctl start nfs-server

# 自動起動
systemctl enable nfs-server

# 準備
exportfs -a
```

# NFSクライアント側の設定
### NFSマウントポイントの作成
```bash
mkdir /yakiniku
```

### NFSマウント設定とマウント
```bash
# ファイル編集
vi /etc/fstab
-----------------
# 末尾に追記
172.31.28.181:/share /yakiniku nfs4 defaults 0 0
-----------------

# マウント
mount /yakiniku

# マウント確認
df

# ファイル作成
touch /yakiniku/test.txt

# サーバ側で確認
ls -l /yakiniku
ls -l /share
```
