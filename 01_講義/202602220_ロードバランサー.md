## SSL オフロード
HTTPS化は信頼性を担保する証明書を利用して通信を暗号化する技術。
WebサイトにSSL証明書をインポートすることで道中の通信を暗号化することができる。

SSL暗号化をロードバランサーにつけるのが一般的。

- SSLオフロード
    - SSL証明書をロードバランサーにインポートし、暗号化処理をロードバランサーに任せること

## パーシステンス
同じ利用者のアクセスを、一定時間ずっと同じサーバへ振り分ける仕組み

### ■ なぜ必要？
ロードバランサーは普通、アクセスをバラバラに振り分けます。

例：
```
1回目 → Web1
2回目 → Web2
3回目 → Web1
```

これだと、
- ログイン状態が消える
- カート情報が消える

という問題が起きます。

### ■ パーシステンスありの場合
```
ユーザA → ずっとWeb1
ユーザB → ずっとWeb2
```
一定時間は同じサーバに接続され続けます。

### ■ 別名
- スティッキーセッション
- セッション維持

### ■ 一言で言うと（試験用）
パーシステンスとは、ロードバランサーが同一ユーザの通信を同じサーバに一定時間固定する仕組みである。
